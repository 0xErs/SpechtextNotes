# SpechtextNotes

## Struktur

- `index.html` - File HTML utama.
- `style.css` - File CSS untuk styling.

## Cara Menggunakan

1. Clone repository ini:
    ```sh
    git clone https://github.com/0xErs/SpechtextNotes.git
    ```

2. Buka file `index.html` di browser.
